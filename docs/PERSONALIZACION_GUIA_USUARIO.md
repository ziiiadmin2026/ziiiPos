# 🎨 Sistema de Personalización - Guía del Usuario

## Tu marca en el sistema

ZiiiPos te permite personalizar completamente la apariencia del sistema con tu propia marca, mientras mantiene el respaldo de **ZIII Solutions**.

---

## 📍 Cómo configurar tu marca

### 1. Accede al módulo de Personalización

Desde el Backoffice:
- Ve a **Backoffice** → **Personalización**
- O directamente: `/backoffice/branding`

### 2. Configura tu logo

**Opción 1: URL del logo**
- Pega la URL de tu logo (debe ser accesible públicamente)
- Formato recomendado: PNG o SVG con fondo transparente
- Tamaño óptimo: 180x48px

**Opción 2: Botón "Subir"** *(próximamente)*
- Sube tu archivo de logo directamente
- El sistema lo almacenará y optimizará

### 3. Nombre de marca (opcional)

Si el nombre comercial de tu restaurante difiere del nombre legal:
- Ingresa tu nombre comercial en "Nombre de marca"
- Ejemplo: Nombre legal "Restaurantes del Norte SA", Marca "El Buen Sabor"

### 4. Personaliza colores

**Color primario**: Se usa en botones, encabezados y elementos principales
- Ejemplo: Color corporativo de tu marca

**Color secundario**: Para acentos, destacados y elementos de acción
- Ejemplo: Color complementario o de contraste

## 💡 Mejores prácticas

### Para logos:

✅ **Recomendado:**
- Fondo transparente (PNG o SVG)
- Alta resolución para pantallas Retina
- Versión horizontal del logo (landscape)
- Colores legibles sobre fondo claro

❌ **Evitar:**
- Logos muy verticales (se verán desproporcionados)
- Fondos de color sólido (prefiere transparente)
- Logos con texto muy pequeño (difícil leer en sidebar)

### Para colores:

✅ **Recomendado:**
- Colores con buen contraste
- Tonos profesionales y legibles
- Coherencia con tu marca física

❌ **Evitar:**
- Colores muy claros (difícil leer texto)
- Combinaciones sin contraste
- Colores fosforescentes o neón

## 🆘 Problemas comunes

### Mi logo no se muestra

**Causa:** URL no accesible o imagen bloqueada
**Solución:** 
1. Verifica que la URL sea pública (no requiera login)
2. Usa HTTPS (no HTTP)
3. Confirma que el archivo existe y no está protegido

### Los colores no se ven bien

**Causa:** Falta de contraste o tonos muy claros
**Solución:**
1. Usa el selector de color para ajustar tonos
2. Prueba colores más oscuros para primario
3. Verifica en diferentes pantallas

### Quiero cambiar el logo después

**Solución:**
- Vuelve a `/backoffice/branding`
- Actualiza la URL del logo
- Guarda cambios
- **Nota:** El cache del navegador puede tardar unos minutos en actualizarse (Ctrl+F5)
